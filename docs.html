Introduction

Get started with ProCSS. This tool is designed to help you design your site easily and quickly to save more time with ready-to-use tools.

Quick Start

This tool gives you a lot of ready-made classes to use in your designs. The classes are prepared according to the first letter of each CSS property.

For example

if you want to use line-height class for your elements, just use the first letter of each word.

lh-1 means.

< span class="lh-1">line height is set to 1.1 in this element< /span>

Cross Browser Compatability

Cross-browser compatibility is the ability of a website or web application to function across different browsers and degrade gracefully when browser features are absent or lacking.

Quick Start

ProCSS cross-check the application for providing the native features across different sets of browsers. If a browser doesn’t support the code then our application should be in a position to provide a different set of code which will run on that particular browser. This will ensure that the application’s basic features are available to the users in any case.

For example

box-sizing CSS property is not supported in all browsers, and to supported you should add all the prefix to special browsers support to work.

Property Prefix

Android: -webkit-

Chrome: -webkit-

Firefox: -moz-

Internet Explorer: -ms-

iOS: -webkit-

Opera: -o-

Safari: -webkit-

Note that most of the properties are supported in most browsers. You can check for unsupported properties from here

Spacing

Quick Start

ProCSS includes a wide range of shorthand responsive margin and padding utility classes to modify an element's appearance.

Usage

The classes are named using the format {property}{sides}-{size} for xs and {property}{sides}-{breakpoint}-{size} for sm, md, lg, and xl.

Where property is one of:

m - for classes that set margin.

p - for classes that set padding.

Where sides is one of:

t - for classes that set margin-top or padding-top.

b - for classes that set margin-bottom or padding-bottom.

l - for classes that set margin-left or padding-left.

r - for classes that set margin-right or padding-right.

blank for classes that set margin or padding on all sides.

Where size is one of:

0 for classes that eliminate the margin or padding by setting it to 0 .

1 - for classes that set the margin or padding to 0.75rem * 1

2 - for classes that set the margin or padding to 0.75rem * 2

3 - for classes that set the margin or padding to 0.75rem * 3

4 - for classes that set the margin or padding to 0.75rem * 4

5 - for classes that set the margin or padding to 0.75rem * 5

6 - for classes that set the margin or padding to 0.75rem * 6

Where breakpoint is one of:

xs - for classes that set margin or padding at extra small screen.

p-xs-0 removes padding at extra small screens.

m-xs-0 removes margin at extra small screens.

ml-xs-2 add margin-left at extra small screens.

sm - for classes that set margin or padding at small screen.

p-sm-0 removes padding at small screens.

m-sm-0 removes margin at small screens.

ml-sm-2 add margin-left at small screens.

md - for classes that set margin or padding at medium screen.

p-md-0 removes padding at medium screens.

m-md-0 removes margin at medium screens.

ml-md-2 add margin-left at medium screens.

lg - for classes that set margin or padding at large screen.

p-lg-0 removes padding at large screens.

m-lg-0 removes margin at large screens.

ml-lg-2 add margin-left at large screens.

xl - for classes that set margin or padding at extra large screen.

p-xl-0 removes padding at extra large screens.

m-xl-0 removes margin at extra large screens.

ml-xl-2 add margin-left at extra large screens.

applied for both margin and padding at all sides.
